Work with Remote IP

Copy local file to remote location

scp filename  root@$ip:/var/www/html

Copy local directory to remote location

scp -r filefolder/  root@$ip:/var/www/html

Apache 403 fobidden

Export and Import db under linux

Export db dvs to file dvs.sql:

# mysqldump -u root -p dvs > dvs.sql

Import dvs.sql to db dvs:

# mysql -u root -p dvs < dvs.sql

Another way:

mysql>use dvs;
mysql>source /home/abc/dvs.sql;

close firewall in fedora19

To let others visit the webapp deployed in my Tomcat, I need to close the firewall:

1.$ service iptables stop
2.$ vi /etc/selinux/config
  # change value of SELINUX to disable
3.$ firewall-config

css combining

1.background
background-position:0 0;  
background-repeat:no-repeat;  
background-attachment:fixed;  
background-color:#f00;  
background-image:url(background.gif);

Can be simplified as:

background: #f00 url(background.gif) no-repeat fixed 0 0;
2.font
font-style:italic;  
font-family:"Lucida Grande",sans-serif;  
font-size:1em;  
font-weight:bold;  
font-variant:small-caps;  
line-height:140%;

Can be simplified as:

font:italic bold small-caps 1em/140% "lucida Grande",sans-serif;
3.list-style
list-style-position:inside;

list-style: url(image.gif) square inside;

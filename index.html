阻止冒泡！！！

window.stopBubble = function (e) {
	//如果提供了事件对象，则这是一个非IE浏览器
	if(e && e.stopPropagation) {
	//因此它支持W3C的stopPropagation()方法
	e.stopPropagation();
} else {
	//否则，我们需要使用IE的方式来取消事件冒泡
	window.event.cancelBubble = true;
}
	return false;
};

Export and Import db under linux

Export db dvs to file dvs.sql:

# mysqldump -u root -p dvs > dvs.sql

Import dvs.sql to db dvs:

# mysql -u root -p dvs < dvs.sql

Another way:

mysql>use dvs;
mysql>source /home/abc/dvs.sql;
